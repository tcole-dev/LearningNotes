## 学习建议

### 学习顺序

```
MySQL 索引 → MySQL 事务/MVCC/锁 → MySQL 性能优化 → MySQL 高可用
    ↓
Redis 数据结构 → Redis 持久化 → Redis 高可用 → Redis 缓存问题 → Redis 分布式锁
    ↓
综合场景设计
```

### 实践建议

1. **搭建环境**：本地搭建 MySQL 主从、Redis 哨兵/集群
2. **源码阅读**：阅读 Redis 源码（如跳表、字典实现）
3. **性能测试**：使用 sysbench、redis-benchmark 进行压测
4. **故障模拟**：模拟主从切换、故障转移场景

### 推荐资源

**书籍：**
- 《高性能 MySQL》
- 《Redis 设计与实现》
- 《深入理解 MySQL 核心技术》

**专栏/课程：**
- 极客时间《MySQL 实战 45 讲》
