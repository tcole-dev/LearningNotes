# 总体阶段划分

```
第1-2周：内存模型（地基）
第3周：类加载机制
第4-5周：GC 全面掌握
第6周：执行引擎 + JIT
第7周：JVM 调优
第8周：源码级理解
```

------

# 🟢 第1周：JVM整体结构 + 运行时数据区

目标：建立宏观结构认知

------

## Day 1-2：JVM整体架构

掌握：

- JVM作用
- 类加载子系统
- 运行时数据区
- 执行引擎

你要能画出：

```
类加载器
    ↓
运行时数据区
    ↓
执行引擎
```

------

## Day 3-4：虚拟机栈（重点）

掌握：

- 栈帧结构
- 局部变量表
- 操作数栈
- 方法调用过程
- 递归为什么会栈溢出

必须能回答：

- 方法调用底层发生了什么？
- 参数如何传递？
- return 如何执行？

------

## Day 5-7：堆 & 方法区

掌握：

- 对象创建流程
- Eden / Survivor
- 老年代
- Metaspace

必须理解：

- 对象什么时候进入老年代？
- 什么是大对象？
- 字符串常量池在哪？

------

# 🟢 第2周：对象模型 + 内存细节

------

## Day 1-3：对象创建全过程

## Day 4-5：对象头结构

理解：

- Mark Word

- 对象布局

------

## Day 6-7：与并发的关系

结合你最近学的：

- CAS
- AQS
- volatile
- 指令重排序

理解：

- happens-before
- 内存屏障
- CPU缓存一致性

# 🟢 第6周：执行引擎 + JIT

# 🟢 第7周：JVM 调优
